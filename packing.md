# 持ち物チェックリスト（日帰りドライブ）

日帰りなので最小限。海辺・屋外散策・長時間運転を想定。

## 必須

- [ ] 免許証
- [ ] 財布 / 現金（小銭・お賽銭用）
- [ ] スマホ + モバイルバッテリー
- [ ] 充電ケーブル（車載用も）
- [ ] ETCカード（高速利用）

## 海辺・屋外散策

- [ ] 歩きやすい靴
- [ ] 羽織れる上着（海風・冷房対策）
- [ ] 日焼け止め
- [ ] 帽子 / サングラス
- [ ] 折りたたみ傘 or レインウェア（雨分岐用）

## あると便利

- [ ] ウェットティッシュ / ティッシュ
- [ ] エコバッグ（マルシェ・お土産用）
- [ ] 常備薬 / 酔い止め
- [ ] 飲み物（車内用）
- [ ] カメラ（写真スポット多め）

## 運転まわり

- [ ] 高速料金 / 駐車場代の用意
- [ ] ナビに目的地を事前登録（二見・VISON・松阪まるよし）
- [ ] 休憩用のSA/PA目星

> 日帰りのため着替え・宿泊用品は不要。荷物は軽めにまとめる。
